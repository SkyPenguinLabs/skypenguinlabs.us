# SkyPenguinLabs

## Production Site

This is a static Cloudflare site. The deployable site lives in `public/`, including the VibeLang section at `/vibelang/`, and the production contract lives in `site.definition.json`.

Use these commands before deploy:

```sh
npm run clean
npm run build
npm run test:visual
```

`npm run build` validates required assets, live route files, and local links. Planned routes can stay in `site.definition.json`, but they should not be linked from rendered navigation until a page exists under `public/`.
