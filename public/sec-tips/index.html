Security Tips

Short defensive notes.

Concise public guidance for triage, automation, documentation, and evidence-minded security work.

Triage

Preserve context first

Capture timestamps, source paths, hashes, and the exact command history before changing an artifact.

Automation

Review before execution

Prefer scripts and runbooks that make intended actions visible before they touch live systems.

Docs

Write evidence, not vibes

Record what was observed, how it was reproduced, and what assumptions remain unresolved.
